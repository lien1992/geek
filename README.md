应用目录
--------

base   基类包

bean   实体包，各模块实体类

constant   常量包

ui   UI界面包,存放UI相关的Activity

 ui.fragment   存放UI相关的Fragment

 ui.adapter   适配器包

 ui.view   自定义UI组件包

util   工具类

 util.http   HTTP操作类

 util.db   数据库操作

 util.sina   新浪微博第三方登录包
